Desenvolvmento de Aplicações Desktop

De momento temos disponível o sistema de gestão de Stock e Venda "Lucros". Este é um produto da Computing Labs que é destinado para quem pretende controlar detalhadamente as actividades de venda e armazém de produtos com ampla visibilidade. Este sistema foi concebido para suportar vários operadores ao mesmo tempo e para trabalhar em rede. De acordo com os requisitos dos nossos clientes, desenvolvemos sistemas do mesmo tipo plataforma para acomodar suas necessidades e exigências.
