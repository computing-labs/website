Curso de Base de dados

O curso de base de dados tém como foco apresentar as técnicas de modelagem de dados, relacionais usando as instruções SQL e a estruturação robusta, regrada do PL/SQL. Este curso possui o total de 48horas a serem ministradas 3 vezes por semana, tendo cada aula a componente teórica e prática em labaratório.
Aproveite a oprutinudade de se proficionalizar nas tecnologias de Base de Dados conhecendo profundamente SQL e PL/SQL em ambiente sqlplus e a regras de modelação de base de dados.

Quem pode participar?

Para aqueles pretendem directamente trabalhar com a análise de sistemas informáticos, concepção de aplicações e regras de sistemas de informação corporativo.
