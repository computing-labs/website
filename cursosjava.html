Curso de Java SE


O curso Java SE ministrado na Computing Labs fornece lhe todas as funcionalidades da linguagem, em ambiente de laboratório apropriado. Usando a IDE Eclipse várias aulas práticas são realizadas para garantir que no final, o saber fazer fique consigo. Este curso oferece todas as ferramentas disponíveis na linguagem Standard desde o inicial ate ao nível mais avançado...

Quem pode participar?

Todos aqueles que tem um gosto pela Informática óptica de desenvolvidor, especialmente programação. O curso fornece conhecimentos sólidos da linguagem e lhe permitirá trabalhar profissionalmente como um verdadeiro programador Java.
